+# Ticket Bot
+
+Простой Discord-бот для управления тикетами поддержки.
+
+## Настройка
+
+1. Склонируйте репозиторий и установите зависимости:
+
+   ```bash
+   pip install -r requirements.txt
+   ```
+
+2. Скопируйте файл `config.example.json` в `config.json` и заполните его своими данными (ID сервера, ролей, категорий и т.д.).
+
+3. Запустите бота:
+
+   ```bash
+   python bot.py
+   ```
+
+## Основные возможности
+
+- Slash-команда `/ticket-panel` публикует панель с кнопкой «Создать обращение».
+- Пользователь выбирает причину через select-меню, затем заполняет форму с подробностями.
+- Бот создает приватный канал тикета, пингует пользователя и роль поддержки, добавляя кнопки для управления.
+- Кнопка «Добавить/Удалить» управляет участниками тикета (нельзя удалить пользователей с ролью поддержки).
+- Кнопка «Закрыть» удаляет тикет с логированием в выделенный канал; для роли поддержки открывается форма с указанием причины закрытия.
